# Contributing to swarm-ai

Thanks for your interest in contributing!

## Getting Started

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Install dependencies: `pip install -r requirements.txt`
4. Make your changes
5. Test your changes
6. Commit with a clear message
7. Open a pull request

## Code Style

- Follow PEP 8
- Use type hints where practical
- Add docstrings for public functions and classes
- Set `PYTHONIOENCODING=utf-8` when running scripts on Windows

## Reporting Bugs

Use the [bug report template](.github/ISSUE_TEMPLATE/bug_report.md) when filing issues.

## Suggesting Features

Use the [feature request template](.github/ISSUE_TEMPLATE/feature_request.md) for new ideas.
