# Security Policy

`swarm-ai` is an experimental project. Only the current `master` branch receives
security fixes; historical experiment artifacts are retained for reproducibility
and are not a supported production interface.

Please report suspected vulnerabilities privately through GitHub's
**Security → Report a vulnerability** flow. Include the affected file or entry
point, a minimal reproduction, impact, and any proposed mitigation. Do not open
a public issue until a maintainer has coordinated disclosure.

Never include API keys, private prompts, personal filesystem contents, or a
production database in a report. Revoke any credential that may have been
exposed before submitting the report.
